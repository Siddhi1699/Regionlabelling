Regionlabelling : Semi-automated Labeling System for Air Quality Data

Welcome to the AQI Data Labeling and Visualization Project! This project involves a web application for visualizing and labeling Air Quality Index (AQI) data. The project is organized into separate folders for backend and frontend components, and includes various documentation files to guide you through the setup and execution of the project.

 Project Structure

- backend/: Contains the server-side code and related files.
- frontend/: Contains the client-side code and related files.
- Pictorial AQI Project Flow.pdf: A visual overview of the project workflow.
- Flowchart.pdf: A detailed flowchart of the project processes.
- Preprocessing flowchart.pdf: Flowchart depicting the preprocessing steps for data.
- UI design basic.pdf: Basic UI design document for the application.
- report.pdf: Detailed documentation of the project.

 Getting Started

 Prerequisites

Ensure you have the following installed:
- Python 3.9
- Node.js and npm (for frontend dependencies)
- Dependencies as specified in the `requirements.txt` in respective folders backend and frontend.

 Backend Setup

1. Navigate to the backend folder:
   
   cd backend

2. Install Python dependencies:

   pip install -r requirements.txt

3. Run the backend server:

   python app.py

 Frontend Setup

1. Navigate to the frontend folder:

   cd frontend

2. Install Node.js dependencies:

   npm install

3. Run the frontend development server:

   npm start

   The frontend application will start and be accessible at `http://localhost:5173/`.
